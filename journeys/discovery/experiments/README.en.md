# Upstream Experiments Directory

New Upstream experiments should be created as directories:

```text
NNN-short-slug/
  experiment.md
  upstream-trail.md
  evidence/
```

Use `experiment.md` for hypothesis, findings, recommendation and Decision Package. Use `upstream-trail.md` for the chronological execution history of the experiment. Store extensive supporting material in `evidence/`.

Flat `*.md` files in this directory are legacy experiments. Do not create new experiments as flat files.
