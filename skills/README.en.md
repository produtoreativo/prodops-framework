# Skills

Skills represent executable behavior used by agents.

Skills are NOT conceptual documentation.
Skills are NOT templates.
Skills are NOT capabilities.

## Skill Structure

Each Skill must contain:
- **Objective** — what the skill does
- **When to use** — entry condition
- **Inputs** — consumed artifacts
- **Outputs** — produced artifacts
- **Capabilities used** — list of capabilities
