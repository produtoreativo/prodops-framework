`prodops/journeys/discovery/upstream-trail.md` is only a high-level index for
milestones between experiments, migrations, promotions, and Upstream process changes in the repository.

# Entry Template

## YYYY-MM-DD HH:MM

### Activity

Describe what happened.

Examples:

- Experiment started
- Prototype updated
- Proof of concept implemented
- Provider documentation reviewed
- Reliability Plan updated
- Experiment completed

### Summary

One or two paragraphs summarizing the work done.

### Artifacts Updated

List only the artifacts updated during this activity.

### Evidence

Reference validation evidence.

Examples:

- Commands executed
- Test results
- Payload examples
- Provider responses
- Evidence files in `evidence/`

### Decision

Choose one:

- Continue experiment
- Start another experiment
- Ready for Assessment
- Discard experiment
- Move to Downstream
- Await business decision
- Await external dependency

### Notes

Additional observations, blockers, or follow-up actions.
